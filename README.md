# Design System

Дизайн-система и кликабельные прототипы, построенные по цепочке
**Figma → токены → GitHub → компоненты → экраны → сценарии**.

Репозиторий хранит дизайн-токены, переиспользуемые компоненты и собранные из
них экраны и сценарии, которые можно открыть в браузере как интерактивный
прототип.

## Структура каталогов

| Каталог        | Назначение                                                                 |
| -------------- | -------------------------------------------------------------------------- |
| `tokens/`      | Дизайн-токены (цвета, типографика, отступы) — источник истины для стилей.   |
| `components/`  | Переиспользуемые UI-компоненты, каждый в своей папке с README.              |
| `prototypes/`  | Кликабельные экраны и сценарии, собранные из компонентов.                   |
| `references/`  | Референсы, выгрузки и материалы из Figma для сверки.                        |
| `assets/`      | Статические ресурсы: иконки, изображения, шрифты.                           |
| `docs/`        | Документация проекта: глоссарий, соглашения, заметки.                       |

## Как посмотреть прототипы

Прототипы — это статические HTML/CSS/JS и публикуются через **GitHub Pages**.
После включения Pages в настройках репозитория прототипы доступны по адресу
вида `https://<owner>.github.io/<repo>/prototypes/`.

Локально достаточно открыть нужный HTML-файл из `prototypes/` в браузере.
